# SUBOLI Design Spec

## Electrical Interfaces

| Interface | Reference | Nets |
|---|---|---|
| DC input panel header | `J1` | `+19V_RAW`, `GND_RAW` |
| Power switch panel header | `SW1` | `+19V_RAW`, `+19V_SW`, `GND_RAW`, `GND_STAR` |
| MA12070 module power | `J5` | `+19V`, `GND_STAR` |
| MA12070 module audio | `J6` | `TACT_INL`, `GND_SIG`, `TACT_INR` |
| Audio input panel header | `J2` | `IN_L`, `IN_R`, `GND_SIG` |
| Headphone panel header | `J4` | `HP_OUT_L`, `HP_OUT_R`, `GND_SIG` |
| Tactile volume panel header | `RV1` | `IN_L`, `IN_R`, `TACT_INL`, `TACT_INR`, `GND_SIG` |
| Headphone volume panel header | `RV2` | `IN_L`, `IN_R`, `HP_INL`, `HP_INR`, `GND_SIG` |

## Power Domains

- `+19V_RAW` enters at `J1` before the panel power switch.
- `+19V_SW` is switched by `SW1`.
- `+19V` is protected and filtered after `F1`, `D1`, `TVS1`, and `C1/C2`.
- `+19V_HP` feeds the OPA1622 headphone branch through `FB1`.
- `VGND` is generated by `U1` for the headphone amplifier input bias.

## Grounding

- `GND_STAR` is the power/star-ground domain.
- `GND_SIG` is the low-level signal return domain.
- `NT1` is the only physical net tie between `GND_SIG` and `GND_STAR`.
- Do not add any other copper short between `GND_SIG` and `GND_STAR`.

## Mechanical Assumptions

- The board is not yet mechanically locked.
- Connector orientation must match the enclosure and panel harness routing.
- The MA12070 module is stacked above or beside the carrier on standoffs, not
  embedded as a PCB footprint on the carrier.
